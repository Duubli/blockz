Blockz
======

Developing
-------------
- Install node packages
```shell
npm install
```

- Install bower packages
```shell
bower install
```

- Watch files and start web server
```shell
npm start
```

- Start socket server
```shell
node server/server.js
```

Building
--------

- To build all files to `./dist/`
```shell
npm run build
```
